Wall Clock

A real-time wall clock built using HTML, CSS, and JavaScript. This project demonstrates how to create a functional analog and digital clock that updates every second using  JavaScript.

🔹 Features
Displays current time (hours, minutes, seconds).
Smooth analog clock movement using CSS rotations.
Digital clock display in HH:MM:SS format.
Fully responsive design.
Built with pure HTML, CSS, and JavaScript – no frameworks or libraries required.
